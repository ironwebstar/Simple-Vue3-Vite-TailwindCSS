# Quickstart template
This is essentially a fresh install of Vue 3 + Vite + Composition API, ESLint, Prettier, TailwindCSS, Vue Router, Vuex

### Project Setup
```
yarn
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```
